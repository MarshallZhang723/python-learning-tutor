# 需求文档：Python 学习辅助助手 (Python Learning Tutor)

## 1. 项目概述
本项目的目标是开发一个基于 Python的Web学习工具。该工具旨在通过交互式练习、语法解释和即时反馈，帮助初学者掌握Python

## 2. 核心功能需求

### 2.1 交互式知识点学习
* **模块化课程**：支持按章节学习（如：变量、数据类型、循环、函数、装饰器）。
* **实时演示**：提供每个概念的代码示例，并允许用户在工具内运行。

### 2.2 智能代码诊断
* **错误分析**：用户输入代码后，工具需利用LLM解释错误原因。
* **优化建议**：除了纠错，还需提供符合 **Pythonic** 风格的改进建议。

### 2.3 练习与测验系统
* **挑战模式**：根据当前进度生成编程题目。
* **自动评分**：通过预设的测试用例验证用户代码的正确性。

## 3. 技术栈要求
* **语言**：Python 3.10+
* **AI 集成**：使用 LangChain 或直接调用 Anthropic API。
* **界面**： Streamlit。
* **数据存储**：使用本地 `json` 或 `sqlite` 存储用户的学习进度和错题集。


## 4. 交互流程设计
### 4.1 详细逻辑步骤
1.  **启动**：显示欢迎界面和学习菜单。
2.  **输入**：用户选择“练习”或“提问”。
3.  **反馈环**：
    * 如果是练习：加载测试用例 -> 捕获用户代码 -> 运行并输出结果。
    * 如果是提问：将代码上下文发送给LLM -> 返回易于理解的中文解释。
